导出备份

将所有进货记录和卖家资料打包成一个 .json 备份文件，下载后发到手机（WhatsApp / 邮件均可），再在手机上导入。

导入备份

上传之前导出的 .json 备份文件，将覆盖当前所有数据。

方法一：上传文件
点击或拖拽备份文件到这里
仅支持 .json 备份文件
方法二：粘贴 JSON 文字（手机推荐）

用手机打开 .json 文件 → 全选文字 → 复制 → 粘贴到下方框框 → 点「从文字导入」

使用步骤
1
电脑上打开 HTML，更新好进货记录
2
点「导出备份文件」，下载 .json 文件
3
把 .json 文件发到手机（WhatsApp 给自己 / 邮件）
4
手机上打开 HTML，到「备份与恢复」→「导入备份文件」，数据立刻同步
